CustomGPT Widget - External Site Integration

About This Example

This demonstrates how to use the CustomGPT widget on an external site with the logo properly included.

The logo (logo.png) is included in the dist/widget directory and will load correctly when you serve these files from a web server.

Embedded Widget Example

Files Required

When deploying to your site, make sure to include these files from the dist/widget directory:

dist/widget/ ├── customgpt-widget.js ├── customgpt-widget.css ├── logo.png └── vendors.js (if using code splitting)

Integration Code

Here's how to integrate the widget:

<!-- Include CSS --> <link rel="stylesheet" href="./customgpt-widget.css"> <!-- Widget Container --> <div id="customgpt-widget"></div> <!-- Include JavaScript --> <script src="./customgpt-widget.js"></script> <!-- Initialize Widget --> <script> const widget = CustomGPTWidget.init({ agentId: 'YOUR_AGENT_ID', containerId: 'customgpt-widget', mode: 'embedded' }); </script>
